==========================================
SunCat Client - 字体系统说明
==========================================

## 如何添加自定义字体

1. 将 TTF 字体文件放入此文件夹
2. 重命名为想要的名字，例如: "myfont.ttf"
3. 在游戏中进入 Fonts 模块
4. 将 "Font" 设置为你文件名（不含扩展名），如 "myfont"

## 推荐免费字体下载

以下字体适合 Minecraft 客户端使用：

1. **思源黑体 (Noto Sans CJK)**
   - 优秀的中文支持
   - 下载地址：https://github.com/google/fonts/tree/main/ofl/notosanscjk
   
2. **思源宋体 (Noto Serif CJK)**
   - 优雅的中文宋体
   - 下载地址：https://github.com/google/fonts/tree/main/ofl/notoserifcjk

3. **Roboto**
   - 干净的现代无衬线字体
   - 下载地址：https://fonts.google.com/specimen/Roboto

4. **Open Sans**
   - 清晰易读
   - 下载地址：https://fonts.google.com/specimen/Open+Sans

5. **Fira Code**
   - 等宽编程字体
   - 下载地址：https://github.com/tonsky/FiraCode

## 系统自带字体（无需下载）

Windows 系统自带的字体可以直接使用，在 Font 设置中输入以下名称：

- "msyh" - 微软雅黑（中文推荐）
- "msyi" - 微软雅黑 Light
- "simhei" - 黑体
- "simsun" - 宋体
- "arial" - Arial
- "segoeui" - Segoe UI
- "tahoma" - Tahoma
- "verdana" - Verdana
- "consolas" - Consolas（等宽）
- "trebuchet" - Trebuchet MS

## Alternate 后备字体

当主字体缺少某些字符时，会使用 Alternate 字体显示：
- 推荐设置为 "msyh"（微软雅黑）以确保中文字符正常显示
- 设置为 "null" 则不使用后备字体
